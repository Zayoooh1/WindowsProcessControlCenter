About

Windows Process Control Center

v0.1.0

Application details

App NameWindows Process Control Center
Version0.1.0
StatusPortable Preview / Release Candidate

Main features

  • CPU priority leveling & adjustment
  • Process freezing & thread suspension controls
  • Executable-level GPU preference management
  • Live process diagnostic listing and search

Safety model

  • Critical Windows system processes are protected and blocked from modifications.
  • Destructive actions (e.g., End Process) require typed keyword confirmations.
  • Administrative check notifies user of potential access restriction issues.

Tech stack

  • Backend: C++ / Win32 API
  • Frontend: HTML5, CSS3, JavaScript (ES6) hosted inside WebView2
  • Bridge: CoreWebView2 asynchronous JSON-based message bridge

Distribution options

  • Portable single executable execution
  • Windows Installer package (MSI)

Known limitations

  • Process freeze/resume controls apply only to the current WPCC session.
  • GPU preference modifications require a target application restart to take effect.
  • Admin privilege level dictates the ability to alter protected system process states.

Rules / Profiles

Manual app-specific presets for process priority and GPU preference.

Profiles v1
Safety note: Profiles are manual in this release. No background automation is active. Destructive actions such as End Process and Freeze are not supported in profiles.
